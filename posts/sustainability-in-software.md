# Sustainability in Software

Something fundamental is shifting in how software gets built and used.

## What Is Actually Changing

The barrier to building software is dropping, but the barrier to building great software is not.

## What This Means

1. **Invest in understanding** — The companies that spend more time on discovery build better products faster

2. **Choose partners over vendors** — Look for teams that care about your outcome

3. **Think in systems** — A collection of features is not a product. A well-designed system is
